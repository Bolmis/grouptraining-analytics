Getting Started

1. Configure Environment Add ZOEZI_API_URL and ZOEZI_API_KEY to your Replit Secrets
2. Add API Routes Edit index.js to add endpoints that fetch data from Zoezi
3. Build Frontend Modify this file (public/index.html) to display your data
4. Read the Docs Check Zoezi API docs.json and the docs/ folder for API reference

Example API Call

// In index.js - add this route: app.get('/api/products', async (req, res) => { const response = await fetch(`${ZOEZI_API_URL}/api/products`); const data = await response.json(); res.json(data); }); // In your frontend JavaScript: const products = await fetch('/api/products').then(r => r.json());
